AI Recommendations Demo

This page demonstrates how the AI button and recommendations system can be easily added to any page on your e-commerce site.

Look for the AI button in the navigation bar above! Click it to see the recommendations dropdown.

Implementation Options

Simple AI Button

The quickest way to add AI features to your site. Just add one line of code to any HTML page:

<script src="js/ai-button.js"></script>

This will:

  • Add an AI button to your navigation automatically
  • Show product recommendations in a dropdown
  • Work without any HTML modifications

Perfect for quickly adding AI features to existing pages.

Full AI Integration

For a complete AI-powered shopping experience, implement the full recommendation system:

<!-- In the head section -->
<link rel="stylesheet" href="styles/navbar-ai.css">

<!-- Before closing body tag -->
<script src="js/ai-recommendations.js"></script>
<script src="js/navbar-ai-integration.js"></script>

Benefits of the full integration:

  • More advanced product recommendations based on browsing history
  • Category affinity calculations
  • Purchase tracking
  • Enhanced personalization over time
  • Better styling integration with your site's theme

This is recommended for production use.
